# Introdução a Cibersegurança

  Segurança cibernética -> Esforço contínuo de proteger indivíduos, organizações e governos
    - Dados pessoais: qualquer informação que possa ser usada para identifica-lo, existindo tanto offline como online
      - Identidade online: 
          - não revelar informações pessoais ao escolher nomes de usuário
      - Seus dados: qualquer informação sobre você
      - Onde estão seus dados -> Os dados ficam armazendos nos mias diversos tipos de lugares, não podendo se ter controle total sobre quem tem acesso a eles
      - A identidade online, suas atividades tudo pode ser registrado e armazenado, sendo de interesse de criminosos, empresas de marketing, mecanismos de pesquisas, entre outros

    - Dados organizacionais: 
        - Dados tradicionais -> Gerados e mantidos por todas as organizações
          - Dados transacionais: Dados de vendas, atividades e operações organizacionais
          - Propriedade intelectual: patentes, marcas, são segredos comerciais.
          - Dados financeiros: Declaração de rendimentos, balanços, fluxos de caixa.

        - Internet das Coisas (IoT) e Big Data 
          - IoT -> rede de objetos físicos que estão ligados à internet, coletando e compartilhando dados.
          
        - O cubo -> Cubo de McCumber
          - 3 dimensões:
          -> Os princípios básicos para proteger os sistemas de informação
            - Confidencialidade: conjuntos de regras que evita divulgação de dados sensívies
            - Integridade: Proteção de informações ou processos contra modificações.
            - Disponibilidade: somente usuários autorizados podem acessar certas informações
          -> A proteção das informações em cada um de seus possíveis estados
            - Processamento: dados usados para executar operações
            - Armazenamento: dados armazenados de forma permanente
            - Transmissão: dados que trafegam entre sistemas
          -> As medidas de segurança usadas para proteger os dados
            - Conscientização, treinamento e educação: medidas para garantir que usuários tenham conhecimentos sobre ameaças a seguranças e formas de proteção
            - Tecnologia: soluções projetadas para proteger sistemas de informação
            - Política e procedimento: controles administrativos sobre como a empresa implementa a garantia de informações
